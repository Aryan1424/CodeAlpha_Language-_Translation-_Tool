It allows users to instantly translate text between multiple languages including:

- Hindi  
- Marathi  
- Tamil  
- Gujarati  
- Bengali  
- French  
- Spanish  

The UI is designed like a simple, attractive Netflix/Amazon-style interface, making it extremely user-friendly.
